# LaunchX

## User Story
As a user, I want to be able to search SpaceX launches to find ones I am interested in;

## Acceptance Criteria:
* Users should be able to search by:
    * Mission name
    * Rocket name
    * Launch year
* Results should include:
    * Mission name
    * Rocket name
    * Launch date
    * Video link
